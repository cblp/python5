Python 3


  >>> adders = [
  ...     lambda x: x + i
  ...     for i in range(3)
  ... ]

  >>> [f(0) for f in adders]
    [2, 2, 2]

Python 5


  >>> let adders = [
  ...     x -> x + i
  ...     for i in range(3)
  ... ]

  >>> [f(0) for f in adders]

  [0, 1, 2]
